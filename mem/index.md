# Project Memory

## Core
Copy convention: Event = real-world fixture/season. Market = tradeable question on an event. Outcome = a side of a market. Section headers like "Featured / Live & upcoming / Browse all" use event(s); "1X2 / BTTS / Top scorer / Player props" use market(s); price rows use outcome.
